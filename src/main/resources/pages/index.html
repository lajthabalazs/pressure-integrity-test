Pressure Integrity Test

Project overview and reference files

What this project does

Pressure Integrity Test is a desktop tool that monitors pressure integrity tests and extrapolates leakage. At the moment, it only support replaying past tests: you load measurement data from ITV (Integrális Tömörségvizsgálat) files, configure the site (containment volume, design pressure, sensor locations), optionally apply calibration, and the application computes averaged pressure, temperature, gas constant (R), and leakage over time.

Files the application uses

  • ITV files – Tab-separated measurement files (date, time, pressures, temperatures, humidity). Timestamps are interpreted as Europe/Budapest.
  • Site config (JSON) – Defines the site id, containment (e.g. net volume), design pressure and leak limits, and locations with pressure, temperature, and humidity sensors. Validated by site-config.schema.json.
  • Calibration (JSON) – Optional mapping from sensor id to linear calibration (real = A × measured + B).
  • Test config (JSON) – Test type and stages (e.g. for EITV), used when running a test.

By default the app uses src/main/resources as the root directory for config files; site configs and calibration can also be chosen via the UI.
